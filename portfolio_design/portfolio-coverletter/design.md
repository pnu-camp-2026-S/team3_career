# 2번 자기소개서형 디자인 기준

## 1. 템플릿 목적

자기소개서형은 사용자의 경험을 기업 제출용 자기소개서 문항 답변으로 정리하는 Word 문서 형식이다.

이 템플릿은 PPT가 아니라 실제 지원서에 가까운 A4 세로 문서로 작성한다. 사용자가 승인한 경험과 Wiki 내용을 바탕으로 문항별 초안을 만들고, 사용자가 직접 수정해 제출할 수 있는 형태를 목표로 한다.

## 2. 전체 구조

자기소개서형은 세로로 쌓이는 표 구조를 사용한다.

```text
[자 기 소 개 서]

[1. 성장과정]
긴 문단 입력 영역

[2. 성격 및 장단점]
긴 문단 입력 영역

[3. 지원동기]
긴 문단 입력 영역

[4. 입사 후 포부]
긴 문단 입력 영역

[5. 핵심 역량 기술]
긴 문단 입력 영역
```

## 3. 제외 항목

다음 항목은 자기소개서형에 포함하지 않는다.

- 회사 로고
- 회사 주소와 연락처
- 이력서 기본 정보 전체
- 학력사항 표
- 경력사항 표
- 자격사항 표
- 병역 정보
- 사진
- 주민등록번호, 주소 등 민감한 개인정보

## 4. 문서 디자인 기준

- 출력 형식은 A4 세로형 Word 문서를 기본으로 한다.
- 상단 중앙에 `자 기 소 개 서` 제목을 배치한다.
- 전체는 큰 표처럼 보이도록 구성한다.
- 각 문항은 `연한 하늘색 제목 바 + 넓은 작성 칸`으로 구성한다.
- 제목 바에는 문항 번호와 문항명만 표시한다.
- 본문 칸에는 긴 문단이 들어갈 수 있도록 충분한 높이를 둔다.
- 테두리는 검정 또는 진회색 얇은 선을 사용한다.
- 장식은 최소화하고 기업 제출용 문서처럼 단정하게 만든다.
- 본문 글자는 읽기 쉬운 기본 문서 폰트를 사용한다.

## 5. 문항별 작성 기준

### 5.1 성장과정

#### 포함 내용

- 가치관이 형성된 배경
- 성장 과정에서 영향을 준 경험
- 현재 지원 직무와 연결되는 태도나 관점

#### 작성 기준

- 단순한 가족사보다 지원자의 가치관과 태도가 드러나야 한다.
- 경험은 직무 역량과 자연스럽게 연결한다.
- 과도하게 감정적인 표현보다 담백한 서술을 사용한다.

### 5.2 성격 및 장단점

#### 포함 내용

- 직무와 연결되는 강점
- 실제 경험에서 드러난 행동 근거
- 보완점과 이를 개선하기 위한 노력

#### 작성 기준

- 강점은 추상적인 성격 표현이 아니라 행동 사례로 설명한다.
- 단점은 치명적인 약점처럼 보이지 않게 하되 숨기지 않는다.
- 보완 노력을 함께 제시해 성장 가능성을 보여준다.

### 5.3 지원동기

#### 포함 내용

- 지원 직무에 관심을 갖게 된 이유
- 본인의 경험과 직무가 연결되는 지점
- 회사 또는 직무에서 기여하고 싶은 방향

#### 작성 기준

- 회사명이 제공되지 않은 경우 임의로 만들지 않는다.
- 막연한 열정보다 직무 중심의 구체적인 동기를 작성한다.
- 사용자의 실제 경험을 바탕으로 지원 이유를 설명한다.

### 5.4 입사 후 포부

#### 포함 내용

- 입사 후 이루고 싶은 목표
- 직무에서 성장하고 싶은 방향
- 보유 역량을 활용한 기여 계획

#### 작성 기준

- 너무 거창한 목표보다 실현 가능한 기여 방향을 작성한다.
- 단기 목표와 장기 방향을 자연스럽게 연결한다.
- 회사명이 없는 경우 특정 회사 내부 정보를 만들어내지 않는다.

### 5.5 핵심 역량 기술

#### 포함 내용

- 핵심 역량 1~3개
- 각 역량을 뒷받침하는 경험
- 문제 상황, 행동, 결과 또는 배운 점

#### 작성 기준

- 역량명만 나열하지 않고 근거 경험을 함께 작성한다.
- 성과 수치가 없으면 임의로 만들지 않는다.
- 지원 직무와 관련 있는 역량을 우선 배치한다.

## 6. AI 작성 원칙

- 사용자가 승인한 경험과 근거 자료를 바탕으로 작성한다.
- 없는 회사명, 직무명, 성과, 수치, 역할은 만들지 않는다.
- 문항별로 500~800자 내외의 자연스러운 문단을 생성한다.
- 자기소개서 문체는 공손하고 자연스러운 서술형으로 작성한다.
- 같은 경험을 반복하더라도 문항 목적에 맞게 강조점을 다르게 둔다.
- 정보가 부족한 문항은 억지로 채우지 않고 보완 필요 항목으로 표시한다.

## 7. 피해야 할 것

- 이력서 내용을 그대로 복사하는 것
- 사진이나 회사 로고를 넣는 것
- 근거 없는 회사명, 성과 수치, 직무 경험을 만드는 것
- 모든 문항을 같은 경험과 같은 문장 구조로 채우는 것
- PPT 카드처럼 보이게 만드는 것
- 문단이 지나치게 길어져 Word 표가 깨지는 것
