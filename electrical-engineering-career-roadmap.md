# 전기공학부 진로 준비 로드맵

전기공학부 학생이 사기업 기술직과 공기업 전기직을 준비할 때 필요한 프로그램 활용 경험, 포트폴리오 예시, 추천 자격증, 핵심 역량을 정리한 문서입니다.

## 1. 전력/전기설비 설계

| 항목 | 정리 내용 |
| --- | --- |
| 필요 프로그램 | AutoCAD, ETAP, MATLAB, Excel, Revit |
| 포트폴리오 예시 | 단선결선도, 부하계산서, 전압강하 계산, 차단기/케이블 선정표 |
| 추천 자격증 | 전기기사, 전기공사기사, 신재생에너지발전설비기사 |
| 핵심 스킬 | 전력계통, 전기설비기술기준, 도면 해석, 계산서 작성 |

### 프로그램 활용 기록

| 프로그램 | 기록하면 좋은 사용 경험 |
| --- | --- |
| AutoCAD | 단선결선도, 분전반 도면, 전기 배선도, 전기실 배치도 작성 |
| ETAP | 부하 흐름, 단락전류, 보호협조 검토 결과 정리 |
| MATLAB | 전압강하, 역률 개선, 부하율, 전력 손실 계산 |
| Excel | 부하계산서, 케이블 선정표, 차단기 용량 산정표 작성 |
| Revit | 건물 전기설비 BIM 모델링, 조명·콘센트·전기기구 배치 |

### 자격증 활용 예시

전기기사를 준비하며 배운 전력공학, 전기설비기술기준, 전기기기 지식을 활용해 부하용량을 계산하고 차단기와 케이블 용량을 선정했다는 식으로 연결하면 좋습니다.

| 추천 자격증 | 전기기사, 전자기사 |
| 핵심 스킬 | 회로해석, 제어공학, PWM, 모터 구동 |

### 프로그램 활용 기록

| 프로그램 | 기록하면 좋은 사용 경험 |
| --- | --- |
| PSIM | Buck/Boost 컨버터, 인버터, 정류기 회로 시뮬레이션 |

### 자격증 활용 예시

전기기사의 회로이론·제어공학 지식을 바탕으로 모터 속도 응답을 분석했고, 전자기사의 회로 해석 지식을 활용해 스위칭 소자의 동작을 검증했다는 식으로 작성하면 좋습니다.

이 분야는 자격증보다 실제 포트폴리오 비중이 큽니다. 전자기사를 적는다면 회로이론, 전자회로, 계측 지식을 활용해 회로를 설계하고 오실로스코프로 측정값을 검증했다는 식으로 연결하는 것이 좋습니다.

## 공기업 전기직 준비

| 단계 | 준비 내용 | 이유 |
| --- | --- | --- |
| 0단계 | 목표 기관 결정 | 한전, 발전사, 전기안전공사, 철도, 지방공기업마다 서류 기준과 전공 범위가 다름 |
| 1단계 | 어학, 한국사, 컴활 등 정량 스펙 확보 | 서류전형에서 기본 점수 또는 가점이 되는 경우가 많아 필기 기회를 얻기 위한 조건이 됨 |
| 2단계 | 전기기사 + 전기공사기사, 즉 쌍기사 준비 | 한전·발전 공기업 전기직에서는 단일 기사보다 쌍기사가 서류 안정성 측면에서 유리 |
| 3단계 | NCS 기본기 시작 | 의사소통, 수리, 문제해결, 자원관리 영역을 시간 제한 두고 푸는 연습 |
| 4단계 | 전공 필기 심화 | 회로이론, 전기기기, 전력공학, 제어공학, 전기설비기술기준 반복 |
| 5단계 | 기관별 기출·모의고사 | 한전형, 발전사형, 전기안전공사형 등 문제 스타일에 맞춰 실전 연습 |
| 6단계 | 면접 준비 | 전공 개념 설명, 안전의식, 공공성, 조직생활 경험, 상황면접 준비 |

## 공기업 목표별 자격증 정리

| 목표 | 우선 자격증 | 추가 스펙 | 준비 방향 |
| --- | --- | --- | --- |
| 한국전력공사 | 전기기사 + 전기공사기사 | 어학, 한국사, 컴활 | 서류 정량 스펙 확보 후 NCS와 전공 필기 병행 |
| 발전 공기업 | 전기기사 + 전기공사기사 | 산업안전기사, 어학, 한국사 | 전기기기, 전력공학, 발전설비, 안전 관련 내용 강화 |
| 한국전기안전공사 | 전기기사 | 전기공사기사, 산업안전기사 | 전기설비기술기준, 접지, 보호장치, 점검 실무 중심 |
| 철도/교통 전기직 | 전기기사 | 전기공사기사, 철도신호기사 선택 | 전력공학, 전기철도, 변전설비, 설비 유지보수 중심 |
| 시설/공단 전기직 | 전기기사 | 소방설비기사 전기, 산업안전기사 | 전기설비 유지관리, 소방전기, 안전관리 중심 |

## 공기업 자격증과 스펙 활용 방식

| 자격증/스펙 | 활용 방식 |
| --- | --- |
| 전기기사 | 전기직 기본 자격증. 전공 지식과 직무 적합성을 보여주는 핵심 자격 |
| 전기공사기사 | 전기공사, 설비, 시공, 현장관리 이해도를 보여주는 자격 |
| 산업안전기사 | 발전소, 설비, 현장 직무에서 안전관리 역량을 보여주는 자격 |
| 한국사 | 공기업 서류 가점 또는 기본 스펙으로 활용되는 경우가 많음 |
| 컴활 | 사무 처리, 데이터 정리, 공공기관 행정업무 역량을 보여주는 보조 스펙 |
| 어학 | 서류 정량 평가에서 중요한 요소. 기준 점수 이상 확보가 필요함 |
| NCS | 서류 이후 필기전형 핵심. 시간 관리와 문제풀이 속도가 중요함 |

## 정리

전기공학부 학생은 사기업과 공기업 준비 방향을 늦어도 3학년 2학기부터 4학년 1학기 사이에 좁히는 것이 좋습니다.

사기업은 프로그램 활용 경험, 프로젝트 깊이, 포트폴리오가 중요합니다. 반면 공기업은 어학·한국사·컴활 같은 정량 스펙, 쌍기사, NCS, 전공 필기가 핵심입니다.

두 방향을 4학년에 동시에 준비하는 것은 현실적으로 부담이 크기 때문에, 본인이 시뮬레이션·시스템 구현형인지, 전공 필기·규정 암기·공기업 시험형인지 먼저 판단하는 것이 중요합니다.
