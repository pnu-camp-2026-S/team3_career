# 마이페이지 및 Supabase 데이터 기준

마이페이지는 추천, AI 분석, 포트폴리오 생성에 필요한 사용자 기본 데이터를 관리하는 곳이다.

## 필수 영역

- 기본 정보: 프로필 사진, 이름, 성별, 생년월일
- 연락처 정보: 이메일, 휴대폰번호, 주소
- 학력: 학교 구분, 학교명, 재학기간, 전공, 부전공/연계전공, 학점
- 희망근무조건: 희망 직무, 세부직무, 업종, 지역, 연봉
- 관심분야: 관심 분야, 관심 기업, 관심 산업

## 데이터 관리 기준

- 마이페이지 입력 데이터는 Supabase를 통해 저장하고 조회한다.
- 프로토타입 단계에서는 모의 로그인 사용자 ID를 기준으로 사용자별 데이터를 구분한다.
- Supabase에는 사용자가 직접 입력하거나 수정한 기본 정보, 연락처 정보, 학력, 희망근무조건, 관심분야를 저장한다.
- 화면의 임시 상태는 `localStorage`를 사용할 수 있지만, 마이페이지의 최종 저장 데이터는 Supabase 기준으로 관리한다.
- 저장 성공, 저장 실패, 불러오기 실패 상태를 화면에 한국어로 안내한다.
- 개인정보가 포함되므로 Supabase RLS(Row Level Security)를 통해 사용자별 데이터 접근 권한을 분리하는 것을 전제로 한다.

## 직무 및 관심분야 선택 기준

- 희망 직무와 관심분야는 chip 형태로 선택/해제할 수 있어야 한다.
- 희망 직무가 바뀌면 더 이상 맞지 않는 세부직무 선택값은 초기화한다.
